# Veroxa — TypeScript Model → PostgreSQL Table Map

All `id` columns become `UUID` primary keys generated by `gen_random_uuid()`.
All tables include `created_at TIMESTAMPTZ NOT NULL DEFAULT now()`.
Tables that track mutations also include `updated_at TIMESTAMPTZ NOT NULL DEFAULT now()`.
