# First-Client Pilot Mode

First-client pilot mode is the current honest operating posture for Veroxa.
It is a guided review and pilot-readiness setup, not a claim that live client
operations are fully connected.

## What is real today

- Public website and Client Demo are available for review.
- Client Portal routes show safe demo/session behavior and empty states when
  live account data is not connected.
- Team Portal routes stay guarded and use calm review-mode language.
- Media, Requests, Updates, and Reports demonstrate the intended Restaurant
  Partner <> Veroxa Team loop.
- Faraz reviews or manually handles anything that would affect a public channel.

## What is sample/demo only today

- Client Demo activity, sample reports, and demo restaurants are sample data.
- Client media tracking may use session/demo records until live account data is
  connected.
- Request and queue previews are review-mode examples unless they are backed by
  an explicitly connected live account.

## What is not connected yet

- No production auth.
- No storage upload for real file bytes.
- No real publishing integration.
- No real reporting data connection.
- No automatic public changes.
- No payment or checkout flow.

## Pilot demo checklist

- Public website remains clean and restaurant-owner friendly.
- Public navigation stays Services / Pricing / Login.
- Client Demo works without promoting old demo wording.
- Team Portal remains locked behind the current team review gate.
- Media flow demonstrates upload, tracker, and status buckets.
- Request flow demonstrates optional client direction.
- Reports demonstrate Weekly Reports and Monthly Reports without invented metrics.
- Updates demonstrate progress, not analytics.
- No fake live data is presented as a connected real account.

## Blockers before a real client account

- Production auth plan and test pass.
- Storage upload path for real photos/videos.
- Client data connection and account scoping.
- Trustworthy reporting data source.
- Manual publishing SOP or an approved publishing integration pass.

## Client-safe explanation

Use: "This is Veroxa's pilot review experience. It shows how media, requests,
updates, and reports will feel. Nothing goes live without Veroxa team review."

Avoid: claims that the system is fully launched, fully connected, or ready for
automatic publishing.
