# First-Client Team Checklist

Status: active manual Team checklist for Faraz/Veroxa.

## Intake checklist

- [ ] Google Business Profile link/access status captured.
- [ ] Website link/access status captured.
- [ ] Facebook page link/access status captured.
- [ ] Instagram profile link/access status captured.
- [ ] Menu/order/reservation links captured.
- [ ] Business hours confirmed.
- [ ] Best sellers captured.
- [ ] Existing offers/promotions confirmed only if already real.
- [ ] Usable media/photos reviewed.

## Manual work checklist

- [ ] Review profile/account consistency across Google, website, Facebook, and Instagram.
- [ ] Review media quality and identify what media is missing.
- [ ] Prepare website alignment/refinement if access is provided and scope is included.
- [ ] Prepare Facebook/Instagram/Google updates from confirmed facts and usable media.
- [ ] Flag any business-truth uncertainty for client confirmation.
- [ ] Prepare weekly update.
- [ ] Prepare monthly report notes without fake metrics or guarantees.

## Safety checklist

- [ ] No live automation/publishing is active.
- [ ] No customer-visible action goes live without Veroxa team review.
- [ ] No Yelp/TikTok/Reels/Ads/live integrations are promised as launch inclusions.
