# First 5 Client Readiness Plan

## Before client 1

- Public offer stable: Complete Online Presence — $495/month.
- Onboarding expectation acknowledgement reviewed.
- Request boundaries tested manually.
- Weekly update and monthly report templates ready.
- Backend SOPs drafted.
- Add-on scopes clear: +$95 new basic website and +$45/profile missing Facebook/Instagram profile.
- No live systems connected.

## What to track for clients 1–5

- Audit fit decision.
- Access checklist status.
- Business-truth confirmation gaps.
- Media supply consistency and quality.
- Portal request categories and 24-hour response quality.
- Weekly update completion.
- Monthly report baseline readiness.
- Online-influenced action signals: calls, directions, website/menu/order clicks, profile actions, content reach, customer mentions, and owner-reported activity.

## Expected manual workflow

Audit → manual review → fit/manual review/not fit → explain offer/boundaries/add-ons/coming soon → onboarding acknowledgement → collect access/media/details → begin weekly update cycle → prepare first monthly report baseline.

## Weekly updates

Weekly updates summarize what Veroxa worked on, what was posted/prepared, what is pending, media needed, confirmations needed, request status, and next focus. No fake metrics and no guarantees.

## Monthly reports

Monthly reports summarize what Veroxa handled, Google/Maps/local visibility, website alignment, Facebook/Instagram content, media working/not-working, reach/action signals, limitations, and next month focus. Internal proof math stays internal.

## Disqualifying issues

- Restaurant expects guaranteed orders/revenue/profit/rankings.
- Restaurant refuses to provide media or confirm business details.
- Heavy customer-service, DMs/comments, refunds/complaints, or order questions are expected.
- Restaurant needs custom development, hosting/domain/email troubleshooting, or advanced technical work as the core need.
- Restaurant demands Yelp/TikTok/Reels/Ads/daily posting/live integrations at launch.

## First-client loyalty discount

The first client receives 20% off for the first 12 months. After 12 months, the discount is retained only while continuously active. If the client leaves and returns later, the discount no longer applies. This is policy/copy only, not payment automation.

## Add-ons

Use +$95 new basic website only when the restaurant needs a simple basic website inside scope. Use +$45/profile missing Facebook/Instagram profile creation only when a launch social profile is missing. Yelp setup is coming soon, not a launch add-on.

## What to learn

- Which onboarding questions cause delays.
- Which media requests produce usable photos.
- Which request boundaries need clearer copy.
- Whether weekly/monthly reporting feels valuable without fake metrics.
- What should be automated later only after manual proof.

## What not to automate yet

Production auth, database, storage, live AI, Google/Meta/Yelp/TikTok integrations, payments, publishing connectors, webhooks, cron jobs, and automated customer-visible execution.
