# SOP — Website Alignment

## Included website alignment scope

- Name/address/phone.
- Hours.
- Menu/order/contact links.
- Google/Facebook/Instagram links.
- Simple description refinement.
- Basic local SEO wording.
- Small content corrections if access is provided.

## +$95 new basic website scope

A simple mobile-friendly restaurant website with name/address/phone/hours, menu/order/contact links, Google/Facebook/Instagram links, basic local SEO wording, and best-seller/service highlights.

## Not included website work

Custom-coded website, advanced design, hosting/domain/email troubleshooting, online ordering setup, speed optimization, plugin troubleshooting, advanced technical SEO, and unlimited pages/edits.

## Confirmation requirements

Confirm exact hours, holiday hours, address, phone, menu items, menu prices if mentioned, order/reservation links, existing offer details, catering, and dietary/health claims before public copy.

## Access requirements

Website access/link is checklist-only in preview/manual mode. Do not connect live systems or make automated changes.

## Documenting changes

Record what was reviewed, what was prepared, what needs confirmation, what access is missing, and what is next for the weekly update/monthly report.

## Escalation and what not to touch

Escalate custom work, domain/hosting/email, plugins, speed, advanced SEO, legal/health claims, and unclear access. Do not touch public website content without confirmation and Faraz review.
