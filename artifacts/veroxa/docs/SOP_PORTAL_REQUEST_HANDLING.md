# SOP — Portal Request Handling

## Included requests

Google update, website alignment, Facebook/Instagram picture post, media guidance, reporting question, menu/contact/link correction, weekly update question, and monthly report question.

## Confirmation-needed requests

Hours, holiday hours, address, phone, menu items, menu prices if mentioned, existing offer/promotion details, catering availability, halal/organic/health claims, order links, and reservation links.

## Add-on requests

- New basic website +$95.
- Missing Facebook or Instagram profile +$45/profile.

## Coming-soon requests

Yelp, TikTok, Reels/video, ads, daily posting, automated publishing, and live integrations.

## Not-included requests

DMs/comments/customer service, refunds/complaints, order questions, full custom development, hosting/domain/email troubleshooting, ad spend changes, and guarantee requests.

## 24-hour response language

“Veroxa will review, answer, or provide the next step within 24 hours. This is not guaranteed completion.”

## Examples

- Included: “Please update our website order link after we confirm it.”
- Confirmation required: “Please mention our holiday hours.”
- Add-on: “We need a basic website.”
- Coming soon: “Can you manage Yelp/TikTok/Reels/Ads?”
- Not included: “Reply to this guest complaint.”

## Escalation rules

Escalate sensitive complaints, unverified claims, discount/offer ambiguity, pricing changes, technical website support, unclear access, and anything outside scope to Faraz.

## No text/call normal workflow

Routine work remains portal-first. Document completed/manual work for weekly updates and monthly reports. No live systems or automated customer-visible execution are connected.
