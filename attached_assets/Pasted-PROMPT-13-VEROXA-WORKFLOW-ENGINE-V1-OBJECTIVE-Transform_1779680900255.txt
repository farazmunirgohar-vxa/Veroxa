PROMPT 13 — VEROXA WORKFLOW ENGINE V1

OBJECTIVE

Transform Veroxa from a collection of dashboards into a functioning operational system.

This phase introduces operational workflows and state transitions using demo data only.

IMPORTANT

DO NOT BUILD:

- Real database
- Real authentication
- Real uploads
- Real AI APIs
- Social media integrations
- Publishing integrations
- Scheduling integrations
- OpenAI integrations

Everything should remain simulated using demo data.

However:

The platform should now behave like a real operating system.
