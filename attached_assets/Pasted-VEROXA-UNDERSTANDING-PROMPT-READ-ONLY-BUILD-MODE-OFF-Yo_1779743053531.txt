WHAT VEROXA IS

Veroxa is a SaaS-style restaurant marketing operating system.

It is not a simple agency website.

It is a structured operating system for restaurant marketing fulfillment.

Core Veroxa flow:

Client uploads real restaurant media
→ System tracks media supply
→ AI assists with review, concepts, and captions
→ Team reviews/refines/approves
→ System schedules posts
→ Posts publish
→ Client is notified
→ Reports update
→ Alerts trigger when content supply runs low

Veroxa should feel premium, dark, clean, structured, and high-trust.

------------------------------------------------

LOCKED ROLES

Client:
Uploads media, views upcoming content, views reports, sends requests, manages account.

Team:
Executes the work: media review, caption drafts, scheduling, weekly reports, alerts.

Operator:
Oversees quality, client health, alerts, failed posts, monthly report approvals, and team oversight.

Owner:
Sees revenue, growth, client risk, retention, critical alerts, and AI/system health.

Strict rule:
Each role should only see what they need.

Client should not see internal operations.
Team should not see owner strategy/revenue.
Operator should not be doing daily execution.
Owner should not be dragged into daily content tasks.
