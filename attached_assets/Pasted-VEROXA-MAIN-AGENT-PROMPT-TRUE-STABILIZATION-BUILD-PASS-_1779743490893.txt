SOURCE OF TRUTH

Veroxa is a SaaS-style restaurant marketing operating system.

Roles:

Client:
Uploads media, views calendar, views reports, sends requests, manages account.

Team:
Executes media review, drafts, scheduling, weekly report validation, and alerts.

Operator:
Oversees client health, quality, alerts, failed posts, monthly report approvals, team oversight, and system status.

Owner:
Sees revenue, growth, client risk, retention, critical alerts, and AI/system health.

Locked Veroxa base pricing:

12-month plan: $997/mo
6-month plan: $1,097/mo
3-month plan: $1,197/mo
No-contract plan: $1,497/mo

No 7-month plan exists.
