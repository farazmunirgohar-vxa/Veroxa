VEROXA CONTEXT

Veroxa is a SaaS-style restaurant marketing operating system for restaurants.

Core flow:

Client uploads media
→ System tracks supply
→ AI assists with review/concepts/drafts
→ Team approves/refines
→ System schedules
→ Post publishes
→ Client is notified
→ Reports update
→ Alerts trigger when content supply drops

Roles:

Client:
Uploads media and views results.

Team:
Executes media review, drafts, scheduling, reports, and alerts.

Operator:
Oversees client health, risk, report approvals, quality, and team oversight.

Owner:
Sees revenue, growth, retention, critical alerts, client risk, and system health.
