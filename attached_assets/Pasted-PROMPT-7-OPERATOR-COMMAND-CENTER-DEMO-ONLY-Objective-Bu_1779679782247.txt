PROMPT 7 — OPERATOR COMMAND CENTER (DEMO ONLY)

Objective:

Build the Veroxa Operator Command Center.

The Operator is responsible for overseeing client health, content flow, reporting completion, onboarding completion, and team workload.

This is NOT an Owner dashboard.

This is NOT a Client dashboard.

This is the operational control layer.

------------------------------------------------

RULES

- Demo only
- No backend
- No database
- No uploads
- No AI APIs
- No social integrations
- No automation
- No scheduling integrations
- No authentication changes
- Reuse existing Veroxa design system
- Maintain dark premium SaaS theme

------------------------------------------------

Create a command dashboard displaying:

- Total Active Clients
- Healthy Clients
- Clients Requiring Attention
- Weekly Reports Pending
- Monthly Reports Pending
- Onboarding Issues
- Media Inventory Warnings
- Content Pipeline Delays

Use realistic sample metrics.
